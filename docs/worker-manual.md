# 매듭 품질 검사 서비스 작업자 매뉴얼

이 문서는 작업자가 태블릿에서 서비스를 사용하는 방법을 안내합니다.

## 1. 작업자용 주요 메뉴

- 메인 검사 화면: `/`
- 출퇴근 기록: `/attendance`
- 공지 확인: `/notices`
- 작업 가이드: `/guides`
- 신고 접수: `/reports`
- 설정: `/settings`

## 2. 메인 검사 화면 사용 방법

작업자는 메인 화면(`/`)에서 아래 순서로 검사를 진행합니다.

1. 작업자 이름을 선택합니다.
2. 검사할 매듭 종류를 선택합니다.
3. 카메라 화면에서 매듭을 중앙에 맞춥니다.
4. `사진 촬영` 버튼을 누릅니다.
5. `검사 시작` 버튼을 누릅니다.
6. 약 1.5초 후 검사 결과를 확인합니다.

## 3. 검사 결과 확인

검사 결과 화면에서는 다음 내용을 확인할 수 있습니다.

- 합격 여부
- 매듭 종류
- 정확도
- 불합격 사유
- 오늘 목표 진행 현황

추가 안내:

- `다시 검사` 버튼을 누르면 처음 단계로 돌아갑니다.
- 합격 시 오늘 목표 달성 수가 자동으로 반영됩니다.
- 음성 안내가 켜져 있으면 합격/불합격 결과가 음성으로 안내됩니다.

## 4. 출퇴근 기록 방법

출퇴근 기록 화면(`/attendance`)에서 다음 순서로 사용합니다.

1. 작업자를 선택합니다.
2. `출근` 또는 `퇴근` 버튼을 누릅니다.
3. 저장 완료 메시지와 기록 시각을 확인합니다.

주의:

- 작업자를 선택하지 않으면 버튼이 동작하지 않습니다.
- 기록 시각은 서버 기준으로 저장됩니다.

## 5. 공지 확인 방법

공지 화면(`/notices`)에서는 작업자별 읽지 않은 공지를 확인할 수 있습니다.

1. 작업자를 선택합니다.
2. 공지 목록을 확인합니다.
3. 원하는 공지를 눌러 상세 내용을 확인합니다.
4. 상세 화면에 들어가면 읽음 처리됩니다.

추가 안내:

- 메인 화면의 `공지` 메뉴 옆 숫자는 읽지 않은 공지 개수입니다.

## 6. 신고 접수 방법

신고 화면(`/reports`)에서 큰 버튼으로 바로 신고할 수 있습니다.

1. 작업자를 선택합니다.
2. 아래 유형 중 하나를 누릅니다.
- 몸 불편
- 재료 부족
- 장비 이상
- 기타
3. `접수되었습니다` 화면이 나오면 완료입니다.

## 7. 작업 가이드 확인 방법

작업 가이드 화면(`/guides`)에서 매듭별 제작 방법을 확인할 수 있습니다.

1. 가이드 목록에서 매듭 종류를 선택합니다.
2. 단계별 사진과 설명을 확인합니다.
3. `이전` / `다음` 버튼으로 단계를 넘깁니다.

## 8. 설정 사용 방법

설정 화면(`/settings`)에서 아래 기능을 조정할 수 있습니다.

- 음성 안내 켜기/끄기
- 음성 속도 조절
- 음성 볼륨 조절
- 휴식 알림 간격 설정

추가 안내:

- 설정값은 브라우저에 저장됩니다.
- 휴식 알림은 출근 상태일 때만 동작합니다.

## 9. 휴식 알림

작업 중 일정 시간이 지나면 휴식 알림이 표시됩니다.

- 기본 메시지: `잠깐 쉬세요! 5분 후 다시 시작해요`
- 알림이 뜨면 카운트다운이 표시됩니다.
- 음성 안내가 켜져 있으면 `휴식 시간입니다`가 함께 재생됩니다.

## 10. 자주 묻는 질문

### Q1. 검사 시작 버튼이 눌리지 않습니다.

다음 항목을 확인합니다.

- 작업자 선택 여부
- 매듭 종류 선택 여부
- 사진 촬영 완료 여부

세 가지가 모두 완료되어야 검사 시작 버튼이 활성화됩니다.

### Q2. 출퇴근 기록이 저장되지 않습니다.

다음 항목을 확인합니다.

- 작업자 선택 여부
- DB 연결 상태
- 네트워크 상태

## 11. 사용 권장 사항

- 태블릿은 검사 시작 전 카메라 권한을 미리 허용합니다.
- 검사 전 매듭이 카메라 중앙에 잘 보이도록 맞춥니다.
- 공지와 가이드는 작업 전에 한 번씩 확인하는 것을 권장합니다.
